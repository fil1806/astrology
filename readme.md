# Астрология: самостоятельное изучение

Учебные материалы и статический сайт по классической астрологии. Материалы пополняются.

## Рекомендации по самостоятельному изучению астрологии

**Первое: изучить все планеты.** Солнце, Луна (светила), Меркурий, Марс, Венера, Сатурн, Юпитер, Уран, Плутон, Нептун, Хирон, Прозерпина, Восходящий и Заходящий лунные кармические узлы, Чёрная Луна и Белая Луна (это фиктивные планеты). Нужно понять эссенциальную природу каждой планеты — её суть саму по себе, до положения в знаке, доме или аспекте. Полезно представлять планеты как живых персонажей: воин Марс, учитель Сатурн, бунтарь Уран.

**Второе: знаки зодиака.** Ещё раз, глубже, понять знаки зодиака и их связь с планетами-управителями. Знак — это «сцена», на которой играет планета; его характер складывается из стихии (Огонь, Земля, Воздух, Вода), креста (кардинальный, фиксированный, мутабельный) и природы планеты-управителя.

**Третье: самое важное — дома гороскопа.** Их 12, по аналогии со знаками (I дом ~ Овен, II ~ Телец и т.д.). Дома показывают, в какой сфере жизни проявляется энергия планеты: личность, деньги, общение, семья, творчество, работа, партнёрство, кризисы, мировоззрение, карьера, друзья, тайное. Для расчёта домов нужно точное время и место рождения.

**Четвёртое: аспекты между планетами.** Тригоны (120°) и секстили (60°) — гармоничные; квадратуры (90°) и оппозиции (180°) — напряжённые; соединение планет (0°) тоже очень важно — энергии сливаются воедино. Напряжённые аспекты — не «плохие»: это мотор развития. Аспекты складываются в конфигурации: большой тригон, тау-квадрат, большой крест.

**Пятое: практика.** Тренироваться «на кошечках»: на друзьях, коллегах и родственниках. Начните со своей карты, затем разберите 10–20 карт людей, которых хорошо знаете, и сверяйте прочитанное с жизнью. Карты можно бесплатно рассчитывать онлайн в программе [sotis-online.ru](https://sotis-online.ru).

Вся эта информация есть в интернете, но при желании можно поучиться на курсах. Пример — Санкт-Петербургская Академия астрологии Константина Дарагана: в целом полезно, но не обязательно.

## Сайт

Статический сайт лежит в папке [`site/`](site/) — чистый HTML/CSS/JS, без сборки и зависимостей. Открыть локально:

```bash
open site/index.html
# или с локальным сервером:
cd site && python3 -m http.server 8000
```

Разместить можно на любом статическом хостинге (GitHub Pages, Netlify, Vercel, обычный веб-сервер) — просто загрузить содержимое `site/`.

### Деплой на Google Cloud Run

В корне лежит `Dockerfile` (nginx + статика, порт берётся из переменной `PORT`, которую задаёт Cloud Run). Деплой одной командой — Cloud Build соберёт образ сам:

```bash
gcloud run deploy astrology-site \
  --source . \
  --region europe-west1 \
  --allow-unauthenticated
```

Проверка локально:

```bash
docker build -t astrology-site .
docker run --rm -e PORT=8080 -p 8080:8080 astrology-site
# открыть http://localhost:8080
```

### Аудио

Исходные записи — голосовые сообщения WhatsApp в формате `.opus` (лежат в корне репозитория). Для сайта они сконвертированы в MP3 через ffmpeg:

```bash
ffmpeg -i "запись.opus" -c:a libmp3lame -q:a 4 site/audio/lecture-N.mp3
```

Новые лекции: сконвертировать так же, положить в `site/audio/` и добавить блок `<div class="audio-item">` в `audio.html`.

### Онлайн-чат (планируется)

Сайт готов к добавлению чата: на каждой странице уже есть кнопка-заглушка (правый нижний угол), которая создаётся в `site/js/main.js` функцией `initChat()`. Чтобы подключить реальный чат, достаточно заменить тело этой функции на код виджета (Tawk.to, Crisp, Jivo — все работают на статических сайтах) или на собственный чат-клиент. Менять HTML-страницы не потребуется.
